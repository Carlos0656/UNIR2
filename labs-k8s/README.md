# Laboratorios de K8s

Los siguientes son una serie de laboratorios para aprender los fundamentos básicos de Kubernetes:

+ Pods.
+ Servicios.
+ Ingress.
+ Network policies.
+ Volúmenes.
+ Service accounts.
+ Secrets.
+ Afinidad y antiafinidad.
